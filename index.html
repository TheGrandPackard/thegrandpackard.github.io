Generate Host Files

To use the Nagios Configuration Writer, simply drag and drop a CSV file containing the hosts that you wish to create configuraton files for. The file format contains the first line with each of the attribute names followed by a line for each host after that. For best results, put the name of the host as the first column as it will also be the name of the file when downloaded.

Example format: host_name,address,user,alias,notificatons_enabled,register
